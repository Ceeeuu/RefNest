# UI Design Guideline

The design language should resemble a museum instead of a dashboard.

Keywords

- Elegant
- Museum
- Gallery
- Art
- Calm
- Warm
- Premium

Avoid

- Material Dashboard
- ChatGPT Layout
- Admin Panel
- Enterprise SaaS

Homepage

A gallery wall displaying artworks.

Artwork Detail

Large artwork on the left.

Artwork information on the right.

Right Sidebar

Permanent AI Curator (OC).

The curator should feel like an NPC inside a museum rather than a chatbot.

New Artwork

The upload page should resemble an artwork registration form rather than a generic CRUD form.
